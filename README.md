# 💪 Gym Website — Fitness Landing Page

A modern, fully responsive gym landing page built with React, TypeScript and Vite — featuring smooth UI, sections for classes, benefits, and contact.

🔗 **Live Demo:** https://gym-pi-eight.vercel.app  
📁 **GitHub:** https://github.com/prem6266/gym

## 🚀 Getting Started

```bash
git clone https://github.com/prem6266/gym.git
cd gym
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 📂 Project Structure

```
/src
  /components  → Reusable UI sections (Hero, Benefits, Classes, etc.)
  /assets      → Images & static files
  /types       → TypeScript type definitions
  App.tsx      → Main app component
  main.tsx     → Entry point
```
